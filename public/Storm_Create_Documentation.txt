STORM CREATE — ПОЛНАЯ ПОЛЬЗОВАТЕЛЬСКАЯ ДОКУМЕНТАЦИЯ
Актуально на: 05.03.2026

============================================================
0) ДЛЯ КОГО ЭТОТ ДОКУМЕНТ
============================================================
Эта инструкция написана для пользователей платформы:
- владельцев ботов;
- администраторов дочерних ботов;
- менеджеров магазина;
- пользователей, которые впервые создают бота через Storm Create.


============================================================
1) ЧТО ТАКОЕ STORM CREATE
============================================================
Storm Create — это конструктор Telegram-ботов.

Через интерфейс платформы можно:
- создать Shop-бота (интернет-магазин);
- создать VPN-бота;
- принимать оплаты (Stars, CryptoBot, ЮMoney и т.д.);
- управлять заказами и клиентами;
- запускать рассылки;
- вести продажи прямо в Telegram.


============================================================
2) ВАЖНЫЕ РОЛИ НА ПЛАТФОРМЕ
============================================================
1. Владелец (Owner)
Создаёт бота, управляет ключевыми настройками и доступами.

2. Администратор дочернего бота
Работает в /admin внутри конкретного бота.

3. Покупатель
Оформляет заказ, оплачивает, получает товар/чек.


============================================================
3) БЫСТРЫЙ СТАРТ ДЛЯ НОВОГО ПОЛЬЗОВАТЕЛЯ
============================================================
1. Откройте главного бота Storm Create.
2. Нажмите “Создать бота”.
3. Выберите тип бота: Shop или VPN.
4. Подключите токен из BotFather.
5. После создания откройте дочернего бота.
6. Введите /admin и настройте его.


============================================================
4) КАК ПОЛУЧИТЬ API-ТОКЕН БОТА В BOTFATHER
============================================================
Пошагово:
1. Откройте @BotFather.
2. Введите /newbot.
3. Укажите имя бота.
4. Укажите username (должен заканчиваться на "bot").
5. Получите токен вида:
   123456789:AA...
6. Скопируйте токен и отправьте его в мастер создания Storm Create.

Важно:
- Токен — это пароль вашего бота.
- Никому его не передавайте.
- Если токен утек, в BotFather используйте /revoke и получите новый.


============================================================
5) КАК СОЗДАТЬ БОТА В STORM CREATE (ПОЛНЫЙ ТУТОРИАЛ)
============================================================
1. В главном боте нажмите “Создать бота” или /newbot.
2. Выберите тип:
   - Интернет-магазин
   - VPN-бот
3. Введите название бота.
4. Вставьте API-токен, полученный в BotFather.
5. При запросе укажите администратора (или оставьте себя).
6. Дождитесь подтверждения “Бот создан”.
7. Откройте ссылку на нового бота.
8. Для настройки введите /admin.

Если токен не принимается:
- проверьте, не вставили ли пробел в начале/конце;
- проверьте, не используется ли токен другим ботом;
- создайте новый токен в BotFather.


============================================================
6) ГЛАВНЫЙ БОТ: ОСНОВНЫЕ РАЗДЕЛЫ
============================================================
Обычно в главном боте используются:
- Создать бота
- Мои боты
- Профиль
- Премиум
- Маркетплейс
- Помощь

В “Профиле” доступны:
- ваш ID;
- баланс;
- пополнение;
- история операций.


============================================================
7) КАК РАБОТАТЬ С "МОИ БОТЫ"
============================================================
Раздел “Мои боты” позволяет:
- открыть карточку каждого бота;
- перейти к настройкам;
- запустить/остановить;
- удалить;
- продлить подписку (если требуется тариф).

Совет:
после создания сразу перейдите в настройки бота и проверьте платежи до запуска рекламы.


============================================================
8) ПЕРВАЯ НАСТРОЙКА SHOP-БОТА (ЧЕК-ЛИСТ)
============================================================
После открытия /admin внутри Shop-бота:
1. Настройки -> Текст приветствия.
2. Категории -> добавить минимум 1 категорию.
3. Товары -> добавить первые товары.
4. Платежи -> включить нужные методы.
5. Чеки -> настроить копии чеков (чат/канал).
6. Сделать тестовый заказ.


============================================================
9) КАТЕГОРИИ И ТОВАРЫ: ПОЛНЫЙ ТУТОРИАЛ
============================================================
9.1 Добавление категории
1. /admin -> Категории.
2. Нажмите “Создать категорию”.
3. Укажите название (и, при необходимости, фото/описание).

9.2 Добавление товара
1. /admin -> Товары -> “Добавить товар”.
2. Заполните шаги: название, описание, цена, фото, категория.
3. Выберите тип товара.

Типы товара:
- physical — физический;
- digital_auto — цифровой с авто-выдачей;
- digital_manual — цифровой с ручной выдачей админом.

9.3 Остатки и ключи
- Для обычного товара задаётся остаток.
- Для digital_auto можно использовать цифровые ключи/контент для выдачи.


============================================================
10) ПЛАТЕЖИ В SHOP-БОТЕ: ПОДРОБНО
============================================================
Путь: /admin -> Платежи

Можно включать/выключать:
1. Telegram Stars
2. CryptoBot
3. ЮMoney
4. QR/Реквизиты
5. Внутренний баланс

10.1 Telegram Stars
- Включите Stars в BotFather.
- Включите Stars в админке платежей.

10.2 CryptoBot
- Должен быть настроен токен у владельца платформы.
- В админке дочернего бота просто включите метод.

10.3 ЮMoney
- Укажите кошелек в настройках платежей.
- Токен API берется из платформы.

10.4 QR/Реквизиты
- Введите реквизиты (текст) и/или загрузите QR-картинку.
- При оплате пользователь нажимает “Я перевел”.
- После этого он обязан отправить скриншот/файл подтверждения.
- Подтверждение автоматически уходит админу.

10.5 Внутренний баланс
- Покупка может списываться с общего баланса пользователя.


============================================================
11) КАК ПОДКЛЮЧИТЬ ЧАТ/КАНАЛ ДЛЯ КОПИЙ ЧЕКОВ
============================================================
Это именно настройка “чеков”, о которой вы просили.

Путь:
1. /admin -> Чеки (или “Чеки и отчеты”).
2. Нажмите кнопку “📢 Чат: ...” (настройка канала/чата чеков).
3. Введите:
   - ID чата/канала, например: -1001234567890
   или
   - username, например: @mychannel
4. Убедитесь, что бот добавлен в этот чат/канал.

Отключение:
- В этом же поле отправьте: off

Если чат/канал чеков не подключен:
- копии чеков отправляются администраторам бота автоматически.

Дополнительно в разделе чеков:
- включение/выключение блоков чека (владелец, поддержка, партнеры);
- стили чеков и предпросмотр (в тарифах Pro/Ultra).


============================================================
12) КАК ПОДКЛЮЧИТЬ КАНАЛ ДЛЯ АВТОПОСТИНГА
============================================================
Это отдельная функция от “чеков”.

Путь:
1. /admin -> Настройки -> “📢 Конфигурация канала”.
2. Нажмите “Установить канал”.
3. Введите ID или @username канала.
4. Бот отправит тестовое сообщение.

Обязательно:
- бот должен быть администратором канала;
- у бота должно быть право отправлять сообщения.

После подключения:
- настройте события для автопубликации (новые товары, заказы, отзывы и т.д.);
- можно делать ручные публикации прямо из админки.


============================================================
13) ОБЯЗАТЕЛЬНАЯ ПОДПИСКА НА КАНАЛ
============================================================
Путь:
1. /admin -> Настройки -> “🔒 Обязательная подписка”.
2. Установите ID канала.
3. Установите ссылку на канал (t.me/...).
4. При необходимости измените текст сообщения (доступ зависит от тарифа).

Важно:
- бот должен быть админом канала, иначе проверка подписки работать не будет.

Отключение:
- в этом же разделе есть кнопка отключения.


============================================================
14) ТРИГГЕР НЕАКТИВНОСТИ (НАПОМИНАНИЯ)
============================================================
Путь:
1. /admin -> Настройки -> “⏰ Триггер неактивности”.
2. Включите триггер.
3. Установите время (в часах), через сколько отправлять напоминание.
4. Отредактируйте сообщение напоминания.

Для чего:
- возвращать неактивных пользователей обратно в бот.


============================================================
15) РАЗДЕЛ "ПОМОЩЬ" ВНУТРИ БОТА
============================================================
Путь:
1. /admin -> Настройки -> “❓ Раздел Помощь”.
2. Добавляйте статьи с заголовком и текстом.
3. Включайте форму обратной связи.

Что получает пользователь:
- удобное FAQ-меню прямо в боте;
- возможность написать в поддержку.


============================================================
16) РАССЫЛКИ
============================================================
Путь:
1. /admin -> Рассылки.
2. Отправьте текст/медиа сообщения.
3. При необходимости добавьте кнопку.
4. Подтвердите отправку.

Рекомендация:
- отправьте сначала тест на себя/админов;
- не отправляйте слишком длинные сообщения без структуры.


============================================================
17) ЗАКАЗЫ И СТАТУСЫ: КАК ЭТО РАБОТАЕТ
============================================================
Для покупателя:
1. Открыть каталог.
2. Добавить товар в корзину.
3. Оформить заказ.
4. Выбрать оплату.
5. Получить чек/статус.

Для админа:
1. Получить уведомление о новом заказе.
2. Если QR/реквизиты — проверить присланный скрин/файл оплаты.
3. Подтвердить оплату.
4. При digital_manual — выдать товар вручную.

При digital_auto:
- выдача может происходить автоматически после подтверждения оплаты.


============================================================
18) ПОПОЛНЕНИЕ БАЛАНСА ПОЛЬЗОВАТЕЛЯ
============================================================
В главном боте:
1. Профиль -> Пополнить баланс.
2. Введите сумму.
3. Выберите способ оплаты.
4. После подтверждения баланс обновится.

В дочернем Shop-боте:
1. Профиль -> Пополнить баланс.
2. Введите сумму.
3. Выберите доступный метод.
4. Подтвердите оплату.

Важно:
- настройки пополнения общего баланса управляются платформой;
- админы дочерних ботов не меняют эти реквизиты по своему усмотрению.


============================================================
19) АНАЛИТИКА, КЛИЕНТЫ, ЭКСПОРТ
============================================================
В админке доступны:
- статистика продаж;
- список клиентов;
- экспорт заказов.

Используйте эти разделы для:
- контроля выручки;
- анализа повторных покупок;
- подготовки отчетности.


============================================================
20) ВЫПЛАТЫ ВЛАДЕЛЬЦУ
============================================================
В соответствующем разделе админки владелец может:
- видеть доступный баланс к выводу;
- создавать заявку на выплату;
- отслеживать статус заявки.

Условия:
- зависят от настроек комиссии/лимитов платформы.


============================================================
21) ИНТЕГРАЦИИ (ЕСЛИ ДОСТУПНЫ)
============================================================
В разделе “Интеграции” можно подключать внешние сервисы
(например, службы доставки/учета), если они активированы в вашем тарифе.

Если интеграция не работает:
- проверьте корректность введенных данных;
- проверьте, есть ли доступ к функции в вашем тарифе.


============================================================
22) ЧТО ДЕЛАТЬ, ЕСЛИ /ADMIN НЕ ОТКРЫВАЕТСЯ
============================================================
1. Убедитесь, что вы администратор именно этого дочернего бота.
2. Проверьте, что бот запущен.
3. Попробуйте /start, потом /admin.
4. Если не помогло — обратитесь к владельцу платформы.


============================================================
23) ЧАСТЫЕ ОШИБКИ И БЫСТРЫЕ РЕШЕНИЯ
============================================================
Ошибка: Payment_provider_invalid
Что значит:
- невалидный токен платежного провайдера карт.
Что делать:
- выбрать Stars/CryptoBot/ЮMoney;
- сообщить владельцу платформы.

Ошибка: “Платеж не найден”
Что делать:
- подождать и нажать “Проверить” повторно;
- проверить корректность метода оплаты.

Ошибка: бот не может писать в канал
Что делать:
- добавить бота админом канала;
- дать право отправлять сообщения;
- проверить ID/username канала.

Ошибка: “токен уже используется”
Что делать:
- сгенерировать новый токен в BotFather.

4. Для QR-оплат сохраняйте подтверждение перевода.
5. Используйте двухфакторную защиту Telegram.

============================================================
28) ДОПОЛНИТЕЛЬНЫЕ ФУНКЦИИ ГЛАВНОГО БОТА
============================================================
1) Реферальная ссылка
- Профиль -> “Реферальная ссылка”.
- Скопируйте ссылку и приглашайте пользователей.

2) История операций (выписка PDF)
- Профиль -> “История операций”.
- Бот отправит PDF с пополнениями, списаниями и покупками.

3) Язык интерфейса
- В главном боте есть переключатель языка.
- По умолчанию используется русский интерфейс.

4) Премиум
- Раздел “Премиум” содержит:
  - Premium аккаунта (лимит активных ботов);
  - подписки для конкретных дочерних ботов (Базовый/Pro/Ultra);
  - разовые платные услуги.

5) Маркетплейс шаблонов
- В главном меню откройте “Маркетплейс”.
- Можно:
  - просматривать публичные шаблоны;
  - покупать платные шаблоны с баланса;
  - применять шаблон к своему Shop-боту.

6) Шаблоны в карточке бота
- Мои боты -> выбрать бота -> Настройки -> Шаблоны.
- Доступно:
  - “Сохранить шаблон” (снимок текущих настроек);
  - “Загрузить шаблон” (применение готового пресета).

7) Авто-напоминание в главном боте (24 часа)
- Если пользователь начал создание бота и не завершил его, система отправляет напоминание через 24 часа.
- В сообщении есть кнопка “Создать бота”.


============================================================
29) ДОПОЛНИТЕЛЬНЫЕ ФУНКЦИИ SHOP-БОТА ДЛЯ ПОКУПАТЕЛЕЙ
============================================================
1) Поиск товаров
- Нажмите “Поиск” и отправьте запрос текстом.
- Бот покажет подходящие товары из каталога.

2) Избранное
- В карточке товара можно добавить/удалить из избранного.
- Раздел “Избранное” показывает сохраненные товары.

3) Промокоды
- В корзине доступен ввод промокода.
- После применения скидка учитывается в финальной сумме.

4) Комментарии к заказу
- На этапе оформления заказа пользователь может оставить:
  - общий комментарий;
  - комментарий по конкретной позиции;
  - медиа-вложения (если предусмотрено сценарием).

5) Профиль в дочернем боте
- В профиле доступны:
  - баланс;
  - пополнение;
  - навигация обратно в меню.


============================================================
30) QR/РЕКВИЗИТЫ: ОБЯЗАТЕЛЬНОЕ ПОДТВЕРЖДЕНИЕ ОПЛАТЫ
============================================================
Логика оплаты по QR/реквизитам:
1. Пользователь выбирает метод “QR/Реквизиты”.
2. Нажимает “Я перевел”.
3. Бот ОБЯЗАТЕЛЬНО запрашивает скриншот или файл-квитанцию.
4. Без файла/скриншота заказ не уходит на подтверждение.
5. После отправки подтверждения:
   - заказ создается в статусе ожидания проверки;
   - скриншот/файл автоматически отправляется администраторам.


============================================================
31) РАСШИРЕННЫЕ НАСТРОЙКИ SHOP-БОТА (/admin)
============================================================
1) Управление функциями
- Вкл/выкл:
  - корзина;
  - отзывы;
  - избранное;
  - цифровые товары;
  - поиск.

2) Редактор кнопок (Pro/Ultra)
- Можно менять названия кнопок главного меню и действий.

3) Редактор инлайн-кнопок (Pro/Ultra)
- Можно менять подписи инлайн-кнопок в карточках товара, оплате, профиле и др.

4) Стиль клавиатуры
- Доступны макеты: классический, современный, сетка, компактный и т.д.
- Есть предпросмотр меню.

5) ИИ-помощник
- В админке можно:
  - включать/выключать ИИ для пользователей;
  - задать свой API-ключ;
  - очистить ключ и использовать системный.


============================================================
32) КАНАЛ, КОММЕНТАРИИ И РАБОЧИЙ ЧАТ
============================================================
1) Канал автопостинга
- Можно включать события:
  - новый товар;
  - новый заказ;
  - смена статуса заказа;
  - новый отзыв.
- Для каждого события можно изменить шаблон сообщения.

2) Ручной пост в канал
- Админ отправляет текст/фото и (опционально) добавляет кнопки.
- Поддерживаются кнопки со ссылками и кнопки перехода к товару.

3) Комментарии под постами канала
- Опция включается в конфигурации канала.
- Под постом добавляется кнопка “Комментарии”, которая ведет к обсуждению в боте.

4) Рабочий чат уведомлений
